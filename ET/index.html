Example:
A simple table with a simple data layout


Code 
import tables from '../src/easy-tables';

tables.CreateTable('#table-1', {
    headers: [
        { col: 4, name: "ID" },

        { col: 4, name: "Age" }

        { id: 1, name: 'Test 1', age: 20 },
        { id: 2, name: 'Test 2', age: 22 },
        { id: 3, name: 'Test 3', age: 25 }
    ]
});

Example with class header property :
You can customize all of the cells with the field and set a custom class, must be defined in the css and accesible from the view currently rendered.


Code 
// custom class exist in the css of the site 
.custom-class {
    background-color: palevioletred;
}

import tables from '../src/easy-tables';

tables.CreateTable('#table-2', {
    headers: [
        { col: 4, name: "ID", class : 'custom-class' },
        { col: 4, name: "Name" },
        { col: 4, name: "Age", class: 'custom-class' }
    ],
    rows: [
        { id: 1, name: 'Test 1', age: 20 },
        { id: 2, name: 'Test 2', age: 22 },
        { id: 3, name: 'Test 3', age: 25 }
    ]
});

Example with title header property :
A simple setter for the title attribute of the cell, usually used for hover ups tool tips, it accepts custom templating.


Code 

import tables from '../src/easy-tables';

tables.CreateTable('#table-3', {
    headers: [
        { col: 4, name: "ID", title : 'This is ID field' },
        { col: 4, name: "Name", title : 'This is the Name field' },
        { col: 4, name: "Age", title : "Current Age : {{GetItemByName this 'age'}}" }
    ],
    rows: [
        { id: 1, name: 'Test 1', age: 20 },
        { id: 2, name: 'Test 2', age: 22 },
        { id: 3, name: 'Test 3', age: 25 }
    ]
});

Example with isarray header property :

To customize more you can have an array of values per cell allowing multiple values and not single ones into an array.

You can also customize the arrays with custom classes and alignment

alignment : 'left' | 'center' | 'right'
class : 'col {any class}'

Also apply custom row classes on the header if you want to change the property of the container row


Code 

import tables from '../src/easy-tables';

tables.CreateTable('#table-4', {
    headers: [
        { col: 4, name: "ID" },
        { col: 4, name: "Groups" },
        { col: 4, name: "Names", isArray: true, rowClass: 'p-0' }
    ],
    rows: [
        { id: 1, name: 'Test 1', names: [{ row: ['John', 'John', 'John'], alignment : 'left' }, { row: ['John', 'John', 'John'] }, { row: ['John', 'John', 'John'] }] },
        { id: 2, name: 'Test 2', names: [{ row: ['John', 'John', 'John'], class: 'col text-danger' }, { row: ['John', 'John', 'John'] }] },
        { id: 3, name: 'Test 3', names: [{ row: ['John', 'John', 'John'], class: 'col border-0' }] }
    ]
});

Example with NthOpt global property :

If you have a fixed table or a selected few rows that one specific classes be applied to them you can added them yourself or dynamically with javascript/

Example : NthOpt: { 1: 'classes', 2: 'classes', 10: 'classes' }

The index count starts at 0, so first row the index would be 0, then 1, 2, 3 etc

Note: the global property NoNthOptInMobile is set to false by default so this properties and classes dont affect mobile view but you can set it true on the global object like NthOpt and it would affect mobile too but could give some undesired results.


Code 
        
import tables from '../src/easy-tables';
        
tables.CreateTable('#table-5', {
    headers: [
        { col: 4, name: "ID" },
        { col: 8, name: "Groups" },
    ],
    rows: [
        { id: 1, name: 'Test 1', },
        { id: 2, name: 'Test 2', },
        { id: 3, name: 'Test 3', },
        { id: 3, name: 'Test 4', },
        { id: 3, name: 'Test 5', },
        { id: 3, name: 'Test 6', },
    ],
    NthOpt:  {1 : 'text-primary', 3 : 'text-danger', 4 : 'bg-success' }
});
