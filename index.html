Header Properties

Header Properties

Example:
A simple table with a simple data layout


Code 
import tables from '../src/easy-tables';
// example 1
tables.CreateTable('#table-1', {
headers: [
{ col: 4, name: "ID" },
{ col: 4, name: "Name" },
{ col: 4, name: "Age" }
],
rows: [
{ id: 1, name: 'Test 1', age: 20 },
{ id: 2, name: 'Test 2', age: 22 },
{ id: 3, name: 'Test 3', age: 25 }
]
});

Example with class header property :
You can customize all of the cells with the field and set a custom class, must be defined in the css and accesible from the view currently rendered.


Code 
// custom class exist in the css of the site 
.custom-class {
    background-color: palevioletred;
}

import tables from '../src/easy-tables';

tables.CreateTable('#table-2', {
headers: [
{ col: 4, name: "ID", class : 'custom-class' },
{ col: 4, name: "Name" },
{ col: 4, name: "Age", class: 'custom-class' }
],
rows: [
{ id: 1, name: 'Test 1', age: 20 },
{ id: 2, name: 'Test 2', age: 22 },
{ id: 3, name: 'Test 3', age: 25 }
]
});

Example with title header property :
A simple setter for the title attribute of the cell, usually used for hover ups tool tips, it accepts custom html template too.


Code 

import tables from '../src/easy-tables';

tables.CreateTable('#table-3', {
    headers: [
        { col: 4, name: "ID", title : 'This is ID field' },
        { col: 4, name: "Name", title : 'This is the Name field' },
        { col: 4, name: "Age", title : '
Current Age : {{GetItemByName this age}}
' }
    ],
    rows: [
        { id: 1, name: 'Test 1', age: 20 },
        { id: 2, name: 'Test 2', age: 22 },
        { id: 3, name: 'Test 3', age: 25 }
    ]
});
